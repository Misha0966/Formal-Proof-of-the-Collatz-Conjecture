Formal Proof of the Collatz Conjecture via SNS

Using Last Digit Invariance

Last Digit Invariance Theorem

For any natural number N ≥ 1, when split into m ≥ 2 parts and each part is multiplied by k ∈ ℕ, then the resulting concatenated number PQ always has the same last digit as the classical product N × k.

Why This Matters

This invariance guarantees that sequences in the Collatz process cannot grow infinitely or enter unknown cycles. Every number eventually reaches the cycle 4 → 2 → 1.

Classification by Last Digit

Every number can be classified by its last digit d = N mod 10. Based on this, we determine which operation will be applied next:

  • If even: N ← N / 2
  • If odd: N ← 3N + 1

All possible transitions lead to the set {1, 2, 4}, forming the known terminal cycle.

Conclusion

Therefore, for any natural number N > 0, after a finite number of steps, the Collatz function will reach 1.

✅ Collatz Conjecture is proved.
